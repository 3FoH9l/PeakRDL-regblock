# PeakRDL-regblock
Generate SystemVerilog RTL that implements a register block from compiled SystemRDL input.

## Installing
Install from [PyPi](https://pypi.org/project/peakrdl-regblock) using pip:

    python3 -m pip install peakrdl-regblock
